GitHubのプライベートリポジトリにプルリクするまで

Mar 14th, 2013

初めてしたことで忘れそうなので.

$ git clone git@github.com:user/repo.git

$ git checkout -b feature

## なんやかんや作業

$ git push origin feature

の後, ブラウザでプルリクすればよい.
